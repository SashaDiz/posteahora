## Troubleshooting Common Issues

### Issue: Can't access https://yourdomain.com

**Solutions**:

1. Check DNS propagation: https://www.whatsmydns.net/ (enter your domain)
2. Check Cloudflare SSL mode is "Full"
3. Wait 10-30 minutes for SSL certificate to issue
4. Check Coolify logs for SSL errors

### Issue: OAuth not working for social media

**Solutions**:

1. Verify callback URLs exactly match in OAuth app settings
2. Make sure you're using HTTPS (not HTTP)
3. Check environment variables are saved in Coolify
4. Redeploy after adding OAuth credentials
5. Check logs: `docker logs postiz-app | grep -i oauth`

### Issue: Database connection errors

**Solutions**:

1. Check DATABASE_URL format is correct
2. Verify DB_PASSWORD matches in DATABASE_URL
3. Restart containers: In Coolify, click "Restart"
4. Check postgres is running: `docker ps | grep postgres`

### Issue: Out of memory / Server slow

**Solutions**:

1. Check resource usage in Coolify
2. Restart containers to free memory

**Solutions**:

## Cost Optimization Tips

### Stay Within $20/month Budget:

1. **Start with local storage** instead of R2 (saves setup time)
2. **Use Resend free tier** (3,000 emails/month is plenty to start)
3. **Don't enable Stripe** unless you're charging users
4. **Monitor usage** in DigitalOcean to avoid overages
5. **Use Cloudflare free tier** (don't upgrade unless you need video)

### When to Upgrade:

- **$18/month (8GB RAM)**: 50-100 active users
- **$24/month (16GB RAM)**: 100-500 active users
- **$48/month (32GB RAM)**: 500+ active users
