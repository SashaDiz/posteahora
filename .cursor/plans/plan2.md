### Step 2.3: Connect to Your Server

**For Mac/Linux**:

1. Open Terminal
2. Type: `ssh root@YOUR_IP_ADDRESS`

                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                - Replace `YOUR_IP_ADDRESS` with your actual IP

3. Type "yes" if asked about fingerprint
4. Enter your password (or it will use SSH key)

**For Windows**:

1. Download and install PuTTY: https://www.putty.org/
2. Open PuTTY
3. Enter your server IP in "Host Name"
4. Click "Open"
5. Login as "root"
6. Enter your password

### Step 2.4: Install Coolify on Your Server

Once connected to your server via SSH, run these commands one by one:

```bash
# Update system packages
sudo apt update && sudo apt upgrade -y

# Install Coolify (automated installer)
curl -fsSL https://cdn.coollabs.io/coolify/install.sh | bash

# Wait 2-3 minutes for installation to complete
```

After installation completes, you'll see a message with:

- Coolify URL: `http://YOUR_IP_ADDRESS:8000`
- Initial admin password

11. **Configure Docker Compose**:

                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                - In the "Docker Compose" field, paste:
```yaml
services:
  postiz-postgres:
    image: postgres:17-alpine
    container_name: postiz-postgres
    restart: always
    environment:
      POSTGRES_PASSWORD: ${DB_PASSWORD}
      POSTGRES_USER: ${DB_USER}
      POSTGRES_DB: ${DB_NAME}
    volumes:
   - postgres-data:/var/lib/postgresql/data

  postiz-redis:
    image: redis:7-alpine
    container_name: postiz-redis
    restart: always

  postiz:
    image: ghcr.io/gitroomhq/postiz-app:latest
    container_name: postiz-app
    restart: always
    ports:
   - "5000:5000"
    depends_on:
   - postiz-postgres
   - postiz-redis
    volumes:
   - postiz-uploads:/uploads

volumes:
  postgres-data:
  postiz-uploads:
```


12. **Add Environment Variables**:

Click on "Environment Variables" tab and add these one by one:

**Required Variables** (Must set these):

```bash
# Database (Generate a strong password!)
DB_USER=postiz_user
DB_PASSWORD=YOUR_STRONG_DB_PASSWORD_HERE
DB_NAME=postiz_db
DATABASE_URL=postgresql://postiz_user:YOUR_STRONG_DB_PASSWORD_HERE@postiz-postgres:5432/postiz_db

# Redis
REDIS_URL=redis://postiz-redis:6379

# JWT Secret (Generate with: openssl rand -base64 32)
JWT_SECRET=YOUR_GENERATED_JWT_SECRET_HERE

# URLs - Using your domain: posteahora.com
MAIN_URL=https://posteahora.com
FRONTEND_URL=https://posteahora.com
NEXT_PUBLIC_BACKEND_URL=https://posteahora.com/api
BACKEND_INTERNAL_URL=http://postiz:3000

# Storage - Start with local, upgrade later
STORAGE_PROVIDER=local

# Production settings
NODE_ENV=production
PORT=3000
```

**Optional but Recommended**:

```bash
# Resend Email (paste your API key from Step 1.4)
RESEND_API_KEY=re_your_key_here

# Cloudflare R2 Storage (from Step 1.5)
STORAGE_PROVIDER=cloudflare-r2
R2_ACCOUNT_ID=your_cloudflare_account_id
R2_ACCESS_KEY_ID=your_r2_access_key
R2_SECRET_ACCESS_KEY=your_r2_secret_key
R2_BUCKET_NAME=postiz-uploads
R2_PUBLIC_URL=https://your-bucket.r2.dev
```

**Save All Variables**
